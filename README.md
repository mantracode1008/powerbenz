# Scrap Business Management System

## Tech Stack
- **Frontend**: React.js (Web), React Native (Mobile)
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Authentication**: JWT

1. Container Entry
2. Item Module
3. Excel Importer
4. Daily Summary
5. Expense / Outcome Module
6. Monthly Reports
7. Charts & Dashboard
8. Master Report
9. Master Data
